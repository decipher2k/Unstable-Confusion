# Unstable-Confusion
A simple GUI for stable diffusion. 8GB GPU RAM are recommended. Less VRAM will lead to higher CPU load and processing time.<br>
Generating an image takes about 2 minutes on an RTX3050 with 8GB VRAM.<br><br>
